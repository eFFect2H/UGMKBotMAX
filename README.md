# UGMKBotMAX (Max.Bot.SDK)

Проект под ТЗ для бота **«Заявки УГМК-Здоровье»** на основе `Max.Bot.SDK` с рабочей прикладной логикой:

- сбор заявки через пошаговый диалог;
- маршрутизация по чатам **по Chat ID**;
- запись заявок в Google Sheets;
- закрытие заявок и отчет по открытым;
- ежедневные напоминания и пометка просроченных.

## Что реализовано

- `RequestOrchestrator` управляет полным циклом заявки.
- `MaxBotGateway` реализует:
  - команды `/start`, `/new`, `/close <guid>`, `/report` (`/отчет`);
  - сценарий пошагового заполнения заявки;
  - формирование текстовых карточек в диспетчерские чаты;
  - отправку подтверждений и напоминаний.
- `GoogleSheetsService` выполняет реальный `append` в Google Sheets API через Service Account.
- `StaticRequestRouter` маршрутизирует заявки по ключу `FacilityType:Direction` в целевой `chatId`.

Фактическая маршрутизация бота делается через `Routing:ChatRouting` в `appsettings.json` и использует ID чатов.

## Конфигурация

Заполните `src/UGMKBotMAX/appsettings.json`:

- `Bot:Token` — токен бота;
- `Routing:FallbackChatId` и `Routing:ChatRouting` — ID целевых чатов;
- `GoogleSheets:SpreadsheetId` и `GoogleSheets:CredentialsJsonPath`.

## Команды бота

- `/start` — приветствие.
- `/new` — создать заявку.
- `/close <guid>` — закрыть заявку.
- `/report` или `/отчет` — список открытых заявок.
